FinAnalyzer

Tally audit toolkit for Chartered Accountants — runs locally in your web browser.

In one line: double-click the Start file for your operating system. It opens FinAnalyzer in your browser. When you're done, double-click the matching Stop file.

What's in this folder

FileWhat it does

Win Stop FinAnalyzer (Windows).batShuts the app down on Windows.

app (folder)All the program files. You don't need to open this — just keep it next to the Start/Stop files.

Before you start — one-time setup

FinAnalyzer needs Node.js (a free, standard runtime) installed once on your computer.

  1. Go to https://nodejs.org and download the LTS version.
  2. Run the installer and accept the defaults.
  3. That's it — you won't need to do this again.

The very first launch also downloads the app's components automatically, so it takes a couple of minutes. Later launches are fast.

Run it

Windows
  1. 1Double-click Start FinAnalyzer (Windows).bat
  2. 2A black window opens and the app loads in your browser at http://127.0.0.1:5173
  3. 3When finished, double-click Stop FinAnalyzer (Windows).bat
macOS
  1. 1Double-click Start FinAnalyzer (Mac).command
  2. 2The app loads in your browser at http://127.0.0.1:5173
  3. 3When finished, double-click Stop FinAnalyzer (Mac).command
First time on Mac: if you see "cannot be opened", right-click the file → Open → Open. You only confirm once.

Using the app

  1. In the browser, upload your Tally export ZIP on the Setup screen.
  2. Pick the months you want, then explore the modules in the left sidebar — Trial Balance, GST registers, TDS, Party Ledger Matrix, and more.
  3. Most modules have an Export to Excel button for working papers.

Everything runs on your own machine. Your data never leaves your computer.
